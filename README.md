# Lock-in

I wanted to get my shit together and keep my habits accountable, and the app to be flexible to my needs. Installation via Sideloadly.

## What it does

- Morning + evening reminders (times are yours to set)
- Tap the notification → quick ✓/✗ checklist for today's tasks
- "Log last night's tasks?" recovery banner if I ghosted the evening ping
- 30-day heatmap so I can see the damage
- Completion line chart + most-skipped-tasks analytics
- Archive tasks instead of deleting — history stays intact

## Install

Sideload via **Sideloadly**. Grab the latest `.ipa` from a release, drop it in Sideloadly, sign with your own Apple ID, plug in your phone, done. Free Apple ID gives you 7 days before it needs re-signing.

## Stack

SwiftUI · SwiftData · Swift Charts · iOS 17+ · fully on-device.
